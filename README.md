# task

Monorepo with API and admin UI.

| Folder | Role |
|--------|------|
| [`task/`](./task) | Backend — Node.js, Express, Sequelize |
| [`ecommerce-backoffice/`](./ecommerce-backoffice) | Frontend — Next.js |

## Quick start

**Backend** (`task/`): install deps, configure DB (see that folder’s config), then `npm run dev`.

**Frontend** (`ecommerce-backoffice/`): `npm install`, `npm run dev` (default port `3253`).
